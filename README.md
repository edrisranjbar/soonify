# Soonify - WordPress Persian Coming Soon Plugin

[![WordPress Version](https://img.shields.io/badge/WordPress-5.0%2B-blue)](https://wordpress.org/)
[![PHP Version](https://img.shields.io/badge/PHP-7.4%2B-purple)](https://php.net/)
[![License](https://img.shields.io/badge/License-GPL%20v2-green)](LICENSE)

A lightweight, clean, and modern Coming Soon plugin for WordPress with full Persian (RTL) language support and local Vazir font integration. Perfect for Iranian WordPress sites under development.

![Soonify Preview](screenshot.png)

## ✨ Features

- 🎨 **Clean & Modern Design** - Beautiful animated coming soon page
- 🇮🇷 **Full RTL Support** - Optimized for Persian/Arabic languages
- 📝 **Local persian Fonts** - No external dependencies, fast loading
- ⚡ **Lightweight** - Minimal code, maximum performance
- 🎛️ **Easy Toggle** - One-click activation/deactivation from admin panel
- 🎨 **Customizable** - Change title, description, and background color
- 📱 **Fully Responsive** - Looks great on all devices
- 🔒 **Admin Access** - Logged-in users can still access the site
- 🔍 **SEO Friendly** - Proper HTTP headers (503 status)
- 🌐 **Translation Ready** - Prepared for multilingual support

## 🚀 Installation

### Automatic Installation (Recommended)

1. Download the plugin ZIP file
2. Go to WordPress Admin → Plugins → Add New
3. Click "Upload Plugin" and choose the ZIP file
4. Click "Install Now" and then "Activate"

### Manual Installation

1. Download and extract the plugin
2. Upload the `soonify` folder to `/wp-content/plugins/` directory
3. Go to WordPress Admin → Plugins
4. Find "Soonify" and click "Activate"


## ⚙️ Configuration

1. Go to WordPress Admin → Soonify
2. Configure your settings:
   - **Enable Coming Soon Mode** - Toggle switch to activate/deactivate
   - **Title** - Main heading text (default: "به زودی...")
   - **Description** - Subheading text (customizable message)
   - **Background Color** - Choose your preferred color
3. Click "Save Changes"

## 🎯 Usage

### Activating Coming Soon Mode
1. Navigate to Soonify settings page
2. Toggle the "Enable Coming Soon Mode" switch
3. Customize the title, description, and background color
4. Save changes
5. Your site will now display the coming soon page to non-logged-in visitors

### Testing the Coming Soon Page
- Open your site in an incognito/private browser window
- Or log out of WordPress to see the coming soon page
- Logged-in administrators will still see the normal site


## 🔧 Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- Write permissions for plugin directory (for font files)
